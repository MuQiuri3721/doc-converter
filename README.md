# 📄 文档转换工具

一个纯前端实现的文档转换工具，支持 Word、PDF、PPT 格式互转。

## ✨ 功能特性

- 🔒 **安全可靠** - 本地转换，文件不上传服务器
- ⚡ **快速转换** - 浏览器本地处理，秒级完成
- 🆓 **完全免费** - 无需注册，无使用限制
- 📱 **响应式设计** - 支持手机、平板、电脑

## 🚀 支持的转换

| 源格式 | 可转换为 |
|--------|----------|
| Word (.docx) | PDF, HTML, TXT |
| PDF (.pdf) | Word, HTML, TXT |
| PPT (.pptx) | PDF, 图片集 |

## 🛠️ 技术栈

- **PDF-Lib** - PDF 文件处理
- **Mammoth** - Word 文档解析
- **PptxGenJS** - PPT 文件生成
- **html2pdf.js** - HTML 转 PDF

## 📖 使用方法

1. 打开网页：https://muqiuri3721.github.io/doc-converter
2. 点击或拖拽上传文件
3. 选择目标格式
4. 点击"开始转换"
5. 下载转换后的文件

## ⚠️ 注意事项

- 纯前端实现，部分复杂格式转换可能有限制
- 大文件转换可能需要较长时间
- 建议在 Chrome、Firefox、Edge 等现代浏览器中使用
