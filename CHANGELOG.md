# 更新日志

## v2.0.1 - Bug 修复版 (2026-02-20)

### 🐛 Bug 修复

1. **HTML 转义问题**
   - 修复了 PDF 转 HTML/Word 时特殊字符显示错误的问题
   - 添加完整的 HTML 实体转义（&, <, >, ", '）

2. **PDF 文本提取优化**
   - 改进空格检测算法，保留原始排版
   - 添加分页进度显示
   - 优化内存管理，及时清理页面资源

3. **PPT 解析增强**
   - 添加幻灯片排序功能
   - 增加错误处理，单张幻灯片失败不影响整体
   - 提升最大处理幻灯片数至 50 张

4. **Word 转 PDF 改进**
   - 添加 mammoth.js 加载检查
   - 改进 html2pdf 调用方式，使用 Promise 包装
   - 添加隐藏元素定位，避免页面闪烁

5. **错误处理增强**
   - 扩展错误类型识别（网络、加密、损坏、超时、内存）
   - 添加错误消息长度限制
   - 改进错误提示的友好性

6. **结果展示优化**
   - 添加结果验证
   - 改进下载链接生成错误处理

### ✨ 其他改进

- 添加全局错误边界
- 改进初始化日志输出
- 优化 CSS 打印样式
- 添加 word-wrap 防止文本溢出
